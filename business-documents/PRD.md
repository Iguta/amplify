# Product Requirements Document (PRD)

## Project Name
**Amplify**  
A personal goal and task tracker web app.

## Overview  
Amplify is designed to help users increase productivity through effective task and goal management, featuring a modern, responsive UI/UX.

## Key Features  
- **Task Management**:  
  - Add, edit, delete tasks with a description, due date, and priority weight.  
  - Visualize tasks in a calendar view, allowing users to click on a date to expand and interact with tasks scheduled for that day.

- **Goal Setting**:  
  - Define personal goals categorized by themes such as Spiritual, Physical Fitness, Academics, and Career.  
  - Manage user-defined categories for both tasks and goals.

- **Progress Tracking**:  
  - Dashboards with charts displaying progress for today, this week, this month, and this year.  
  - Simple, clear progress reports to encourage user engagement and motivation.

## UI/UX Design  
- **Modern and Minimalistic**:  
  - Highly polished design focusing on clarity and calm to enhance productivity.
  - Default dark theme with soft accent colors.

- **Responsive Design**:  
  - Fully functional across mobile, tablet, and desktop devices.

- **Calendar-First Layout**:  
  - Smooth day expand/zoom interactions for intuitive user engagement.  
  - Clean task cards with subtle animations.

## Additional Considerations  
- No authentication required; the app is designed for personal use only.

# Amplify

Amplify is a personal goal and task tracker web app designed to help users increase productivity through effective task and goal management, featuring a modern, responsive UI/UX.

## Product Overview
Amplify provides a calendar-first experience for managing tasks and setting goals with progress tracking dashboards.

## Key Features
- **Task Management**
  - Add, edit, and delete tasks with description, due date, and priority weight.
  - Calendar view with day expansion to interact with tasks scheduled for that date.
- **Goal Setting**
  - Define goals by themes (Spiritual, Physical Fitness, Academics, Career).
  - Manage user-defined categories for both tasks and goals.
- **Progress Tracking**
  - Dashboards with charts for today, this week, this month, and this year.
  - Simple, clear progress reports to encourage engagement.

## UI/UX
- Modern, minimalistic, polished design
- Default dark theme with soft accent colors
- Responsive across mobile, tablet, and desktop
- Calendar-first layout with smooth day expand/zoom interactions

## Tech Stack
- React + TypeScript (Vite)
